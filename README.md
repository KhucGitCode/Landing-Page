# Responsive Landing Page

A fully responsive landing page built using **HTML and CSS**, designed to represent a real-world product/service.  
This project focuses on clean layout structure, mobile-first design, and manual responsiveness without using any CSS framework.

## Project Overview

This landing page includes all essential sections commonly found in modern websites:
- Hero section with a clear call-to-action
- Features section highlighting key benefits
- Pricing plans for different user needs
- Testimonials to build trust
- Final CTA section for user engagement

The project was built to practice **core frontend fundamentals** and responsive design principles.

---

## Tech Stack

- **HTML5**
- **CSS3**
- Flexbox
- CSS Grid
- Media Queries (No Bootstrap / No Tailwind)

---

## Features

- Fully responsive design (mobile, tablet, desktop)
- Mobile-first layout approach
- Clean and reusable CSS
- Structured sections with semantic HTML
- Modern UI layout using Flexbox and Grid

---

## Responsiveness

The layout adapts smoothly across different screen sizes:
- **Mobile:** Stacked layout for better readability
- **Tablet:** Adjusted spacing and alignment
- **Desktop:** Multi-column layout for better visual hierarchy

Media queries were written manually to handle responsiveness.
